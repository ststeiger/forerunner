Namespace: toolPane

$.forerunner. toolPane

ToolPane widget used with the reportViewer
Properties:
Name Type Description
options object The options for toolPane
Properties
Name Type Description
$reportViewer Object The report viewer widget
toolClass String The top level class for this tool (E.g., fr-toolpane)
Source:
  • ReportViewer/js/ToolPane.js, line 15

Example

$("#toolPaneId").toolPane({
 $reportViewer: $viewer,
 toolClass: "fr-toolpane"
});

Note:
 ToolPane can be extended by calling the addTools method defined by $.forerunner.toolBase
