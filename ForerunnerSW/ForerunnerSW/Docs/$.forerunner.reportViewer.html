$.forerunner. reportViewer

Widget used to view a report
Properties:
Name Type Description
options Object The options for reportViewer
Properties
Name Type Description
reportViewerAPI String Path to the REST calls for the reportViewer
pingInterval Integer Interval to ping the server. Used to keep the sessions active
toolbarHeight Number Height of the toolbar.
pageNavArea Object jQuery selector object that will the page navigation widget
paramArea Object jQuery selector object that defineds the report parameter widget
DocMapArea Object jQuery selector object that defineds the Document Map widget
userSettings Object User settings used for user specific options
onInputBlur Function Callback function used to handle input blur event
onInputFocus Function Callback function used to handle input focus event
$appContainer Object Report container
parameterModel Object Parameter model
savePosition Object Saved report page scroll position
viewerID String Current report viewer id.
rsInstance String Report service instance name
showSubscriptionUI String Show Subscription UI if the user has permissions. Default to false.
zoom String Zoom factor, default to 100.
exportCallback function call back function for all exports, will call instead of window.open
printCallback function call back function for print, will call instead of window.open

Example

$("#reportViewerId").reportViewer();
$("#reportViewerId").reportViewer("loadReport", reportPath, 1, parameters);
