Mobilize the power of Microsoft’s SQL Server Reporting Services on any device.

Forerunner Software unlocks and extends the power of Microsoft’s SQL Server by allowing users access to core business intelligence across a broad range of communication devices. A solution built and designed for IT and Developer professionals: Using existing code and maintaining corporate standards to protect critical data.

With this out of the box solution you can be up and running within minutes—making you the hero for your company by providing access to key business information to speed critical decisions. Forerunner is an easy to use solution that extends the power of your existing investments; eliminating the need for additional development or user training.

 
Mobile Optimized Solution

Forerunner Software is designed to get you up and running in minutes. It is a fully functional, out of the box experience displays data dynamically across the browser and device of choice.

  • Supported on IOS, Android and Windows mobile
  • Browser agnostic (IE8+, Chrome, Safari, Firefox)
  • Fully localized, multi-language UI

 
Business Intelligence Anywhere. No Training Required.

Corporate mobility is a growing trend and concern. Companies with a mobile platform for reporting and analytics can make critical business decisions faster. Forerunner Software displays data responsively and retains user access, so everyone sees the data they’re working with.

  • Familiar display and easy to use interface
  • New end user features: Favorites, recent history, and personalized parameters


 
Security & Information Protection.

Forerunner Software integrates with your company’s exiting security policies. You control who has access using standard SQL Server methods.

  • Integrates with Windows NTML and Kerberos
  • Supports basic and forms based authentication
  • Optional SSO provider

 
Rapid Application Development

Forerunner takes advantage of IT and developer’s skills and investments in Microsoft’s SQL Server development platform. Easy to integrate into modern applications and the fluid layout means that content adapts to all form factors. Forerunner reduces infrastructure costs and helps improve administrator efficiency.

  • Natively plugs into your HTML5 and MVC applications
  • Uses industry standard interfaces; REST, jQuery
  • Completely customizable via code and/or simple CSS changes
  • Easy to integrate and extend with an extensive API and documentation
