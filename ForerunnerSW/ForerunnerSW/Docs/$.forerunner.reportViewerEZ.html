Namespace: reportViewerEZ

$.forerunner. reportViewerEZ

Widget used to view a report
Properties:
Name Type Description
options Object The options for reportViewerEZ
Properties
Name Type Description
DefaultAppTemplate Object - The helper class that creates the app template. If it is null, the widget will create its own.
navigateTo Object Callback function used to navigate to a selected report. Only needed if isReportManager == true.
historyBack Object Callback function used to go back in browsing history. Only needed if isReportManager == true.
isReportManager Boolean A flag to determine whether we should render report manager integration items. Defaults to false.
isFullScreen Boolean A flag to determine whether show report viewer in full screen. Default to true.
userSettings Boolean Custom user setting
rsInstance String Report service instance name
useReportManagerSettings Boolean Defaults to false if isReportManager is false. If set to true, will load the user saved parameters and user settings from the database.
toolbarConfigOption Boolean Defaults to forerunner.ssr.constants.toolbarConfigOption.full
handleWindowResize Boolean Handle the window resize events automatically. In cases such as dashboards this can be set to false. Call resize in this case.
showBreadCrumb Boolean A flag to determine whether show breadcrumb navigation upon the toolbar. Defaults to false.
zoom- String Zoom factor. Defaults to 100.

Example

$("#reportViewerEZId").reportViewerEZ({
 DefaultAppTemplate: null,
 navigateTo: me.navigateTo,
 historyBack: me.historyBack
 isReportManager: false,
 userSettings: userSettings
});
