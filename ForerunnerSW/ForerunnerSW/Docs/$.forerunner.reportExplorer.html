Namespace: reportExplorer

$.forerunner. reportExplorer

Widget used to explore available reports and launch the Report Viewer
Properties:
Name Type Description
options Object The options for reportExplorer
Properties
Name Type Description
reportManagerAPI String Path to the report manager REST API calls
forerunnerPath String Path to the top level folder for the SDK
path String Path passed to the GetItems REST call
view String View passed to the GetItems REST call
selectedItemPath String Set to select an item in the explorer
$scrollBarOwner Object Used to determine the scrollTop position
navigateTo Object Callback function used to navigate to a selected report
$appContainer Object The container jQuery object that holds the application
explorerSettings Object Object that stores custom explorer style settings
rsInstance String Report service instance name
userSettings Object User settings used for user specific options
onInputBlur Function Callback function used to handle input blur event
onInputFocus Function Callback function used to handle input focus event

Example

$("#reportExplorerId").reportExplorer({
   reportManagerAPI: "./api/ReportManager",
   forerunnerPath: "./forerunner/",
   path: "/",
   view: "catalog",
   navigateTo: navigateTo,
   $appContainer: me.$container,
   explorerSettings: explorerSettings
});
