Optimized Responsive Application

Forerunner Mobilizer is designed to get you up and running in minutes. It is a fully functional, out of the box experience that displays data dynamically across all browsers and mobile devices.

  • Browser agnostic (IE8+, Chrome, Safari, Firefox)
  • Fully localized, multi-language UI
  • Supported on IOS, Android and Windows mobile

Security & Information Protection

Forerunner Mobilizer integrates with your company’s existing security policies. You control who has access using standard SQL Server methods.

  • Integrates with Windows NTLM and Kerberos
  • Supports basic and forms based authentication
  • Optional SSO provider
